Minor Project Overview – AI Resume Screening System

Project Overview

The AI Resume Screening System is a web-based application developed to automate the process of resume shortlisting. It compares uploaded resumes with a job description and ranks candidates based on relevance and skill match.

Objectives

  • To reduce manual effort in resume screening.
  • To rank candidates using NLP and machine learning concepts.
  • To provide an easy-to-use recruiter dashboard.
  • To generate downloadable shortlist reports.

Key Features

  • Upload multiple resumes in PDF format
  • Enter job description for comparison
  • Automatic resume ranking and score generation
  • Skill extraction from candidate resumes
  • Graphical candidate score visualization
  • Downloadable shortlist report

Technology Stack

  • Frontend: HTML, CSS
  • Backend: Python Flask
  • ML/NLP: TF-IDF, Cosine Similarity
  • Libraries: scikit-learn, matplotlib, pandas, PyPDF2

Working Principle

The system extracts text from PDF resumes, converts both the job description and resumes into numerical vectors using TF-IDF, and then calculates similarity using cosine similarity. Candidates are ranked based on the generated score.

Future Scope

  • Advanced NLP-based skill extraction
  • Database integration for candidate history
  • Recruiter login system
  • Resume classification using ML models
  • ATS compatibility analysis
